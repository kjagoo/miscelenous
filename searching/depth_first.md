The implementation below uses the stack data-structure to build-up and return a set of vertices that are accessible within the subjects connected component. Using Pythonâ€™s overloading of the subtraction operato

def dfs(graph, start):
    visited, stack = set(), [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)
    return visited
